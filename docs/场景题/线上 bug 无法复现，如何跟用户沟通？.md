# 线上 bug 无法复现，如何跟用户沟通？

> 沟通核心：共情 + 明确诉求 + 提供临时方案 + 持续跟进

您好！非常抱歉给您带来不好的体验，我们非常重视您反馈的这个问题。目前我们在测试环境暂时无法复现这个问题，为了尽快定位并解决，想向您确认几个信息：

- 您遇到问题时的操作步骤（如：点击 XX 按钮 → 输入 XX 内容 → 出现 XX 现象）；
- 您使用的浏览器型号 / 版本、设备类型（PC / 手机）、系统版本；
- 问题出现的频率（必现 / 偶尔）、是否有报错提示（如弹窗、空白）；
- 方便的话，能否提供问题出现时的截图 / 录屏？

同时，我们先为您提供一个临时解决方案：[如：清除浏览器缓存 / 切换浏览器重试 / 重启设备]，您可以先尝试。我们会根据您提供的信息优先排查，有进展会第一时间同步您，感谢您的配合！
